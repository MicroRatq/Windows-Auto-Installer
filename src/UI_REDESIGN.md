## 使用方法

### 启动应用
```bash
cd src
npm start
# 或
npm run dev  # 开发模式
```

### 操作说明
1. **切换功能模块**：点击顶部主菜单（镜像、系统配置、软件安装）
2. **选择具体功能**：点击左侧二级菜单
3. **调整侧边栏**：拖动左侧边缘的分界线
4. **切换主题**：点击标题栏右侧的主题按钮
5. **窗口控制**：使用右上角的三个按钮

## 技术实现

- **无边框窗口**：Electron `frame: false`
- **自定义标题栏**：HTML + CSS实现
- **可拖动分界线**：JavaScript鼠标事件处理
- **主题系统**：CSS变量 + data-theme属性
- **Windows 11风格**：参考Windows 11设计规范
